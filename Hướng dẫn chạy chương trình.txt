Cài Python (tích “Add Python to PATH”), cài VS Code

Mở thư mục dự án trong VS Code → tạo venv:

py -m venv .venv (Win) → kích hoạt: .venv\Scripts\activate

(macOS/Linux) python3 -m venv .venv → source .venv/bin/activate

Chọn interpreter .venv (Ctrl+Shift+P → “Python: Select Interpreter”).

Cài thư viện: pip install numpy matplotlib opencv-python scikit-image
